Minar Technologies

Leveraging Learning, Teaching and Research through international co-operation

MinarTech is a hub for international, community driven projects aiming at Curated, Thematic Services for Cultural Heritage. The outcome of MinarTech Projects will be published in their respective, separate Project Sites. Prototypes may be published here. Code and Resources of will be shared on GitHub. Co-operation between technical developers and curators will be facilitated using the Discourse forum software.

First Project: YT-iFrame

Displaying synchronized text in iFrame using the YouTube API. A prototype application to support the teaching and learning of, initially, Hindi/Urdu and Farsi.

The code to the project can be found at https://github.com/minar-tech/YT-iFrame/
