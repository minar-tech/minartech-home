Adaptation for sign-by-sign transliteration

This is work in progress. The aim is to arrive at a system that enables computerized, lossless, intuitively human-readable, two-way transliteration using only one Latin character for each Persian letter. The preference is to arrive to a system that is as close as possible to transliteration of Arabic. The primary aim is not to support pronounciation. This will be implemented as a separate layer based on a different encoding and a different set of rules.
